### 注入方式选择指南

| 场景 | 推荐方式 | 理由 |
|------|----------|------|
| 必需依赖 | 构造函数注入 | 不可变性、强制依赖 |
| 可选依赖 | Setter注入 | 灵活性、可重新注入 |
| 测试类 | 字段注入 | 简洁性 |
| 复杂对象 | @Bean配置 | 精细控制创建过程 |

### 常见问题快速解决

| 异常类型 | 快速解决方案 |
|----------|-------------|
| NoSuchBeanDefinitionException | 检查注解、扫描路径、@Bean定义 |
| NoUniqueBeanDefinitionException | 使用@Qualifier或@Primary |
| 循环依赖 | 使用@Lazy或重构设计 |

## 🛠️ 实用工具和技巧

### 1. 快速检查Bean信息

```java
@Component
public class BeanInfoTool implements CommandLineRunner {

    @Autowired
    private ApplicationContext context;

    @Override
    public void run(String... args) {
        // 列出所有Service类型的Bean
        Arrays.stream(context.getBeanNamesForType(Object.class))
              .filter(name -> name.toLowerCase().contains("service"))
              .forEach(System.out::println);
    }
}
```

### 2. 调试依赖注入

```properties
# 启用Spring调试日志
logging.level.org.springframework.beans.factory=DEBUG
logging.level.org.springframework.context=DEBUG
```

### 3. 常用注解速查

```java
// Bean定义注解
@Service      // 业务逻辑层
@Repository   // 数据访问层
@Component    // 通用组件
@Controller   // 控制器层

// 依赖注入注解
@Autowired    // Spring自动注入
@Inject       // JSR-330标准注入
@Resource     // JSR-250标准注入

// 配置注解
@Configuration // 配置类
@Bean         // 定义Bean
@Value        // 注入配置值

// 条件注解
@ConditionalOnProperty  // 基于属性条件
@ConditionalOnClass     // 基于类存在条件
@ConditionalOnBean      // 基于Bean存在条件
```
