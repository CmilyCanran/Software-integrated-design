# Spring Boot 学习笔记

> **系统化学习Spring Boot，从核心概念到实战应用**

## 🎯 学习理念

### 🏗️ 概念优先，实践并重
我们采用**概念驱动**的学习方法：
1. **先理解核心思想** - 掌握Spring的设计哲学
2. **再学习具体技术** - 在理解基础上学习Spring Boot
3. **最后实战应用** - 通过项目巩固所学知识

### 🔄 学→练→会 循环学习
每个知识点都遵循这个循环：
- **学** - 理论学习，理解概念
- **练** - 动手实践，编写代码
- **会** - 总结提升，真正掌握
