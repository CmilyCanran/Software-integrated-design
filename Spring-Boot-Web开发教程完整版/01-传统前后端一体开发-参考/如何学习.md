# 🎓 如何使用本教程学习Spring Boot Web开发

## 📖 学习路径指南

### 🚀 开始之前

#### 环境准备清单
- [ ] **Java 21 JDK** - 确保安装并配置JAVA_HOME
- [ ] **Maven 3.6+** - 安装并配置阿里云镜像源（可选）
- [ ] **MySQL 8.0** - 安装并启动服务
- [ ] **IntelliJ IDEA** - 推荐使用Ultimate版本
- [ **Git** - 用于版本控制

#### 基础知识要求
- Java基础语法熟练掌握
- 面向对象编程概念理解
- SQL基础语句了解
- HTML/CSS/JavaScript基础

4. **求甚解** - 不仅知其然，更要知其所以然

## 🛠️ 开发环境配置

### 1. IDEA配置

```xml
<!-- 推荐安装的IDEA插件 -->
<plugins>
    <plugin>Lombok</plugin>
    <plugin>Spring Boot Helper</plugin>
    <plugin>Database Navigator</plugin>
    <plugin>RESTful Toolkit</plugin>
    <plugin>GitToolBox</plugin>
</plugins>
```

### 2. Maven配置

```xml
<!-- settings.xml 阿里云镜像 -->
<mirrors>
    <mirror>
        <id>aliyunmaven</id>
        <mirrorOf>*</mirrorOf>
        <name>阿里云公共仓库</name>
        <url>https://maven.aliyun.com/repository/public</url>
    </mirror>
</mirrors>
```

### 3. 数据库配置

```sql
-- 创建数据库
CREATE DATABASE springweb CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户
CREATE USER 'springweb'@'localhost' IDENTIFIED BY 'springweb123';
GRANT ALL PRIVILEGES ON springweb.* TO 'springweb'@'localhost';
FLUSH PRIVILEGES;
```

## 🔍 常见问题解决

### 1. 项目启动失败
```bash
# 检查步骤
1. 检查Java版本是否正确
2. 检查Maven依赖是否下载完成
3. 检查数据库连接配置
4. 检查端口是否被占用
```

### 2. 数据库连接问题
```bash
# 排查步骤
1. 检查MySQL服务是否启动
2. 检查数据库用户权限
3. 检查防火墙设置
4. 检查连接字符串格式
```

### 3. 依赖冲突问题
```bash
# 解决方法
1. 使用Maven依赖树查看冲突
2. 排除冲突依赖
3. 强制指定版本
4. 更新Spring Boot版本
```
