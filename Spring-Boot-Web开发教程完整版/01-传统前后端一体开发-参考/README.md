# Spring Boot Web开发完整教程

## 📚 教程概述

本教程将带您从零开始，系统学习Spring Boot Web开发的全过程。我们将构建一个完整的用户管理系统，涵盖从前端页面到后端API，再到数据库持久化的全栈开发技能。

## 🎯 学习目标

- 掌握Spring Boot项目的基础配置
- 学会使用Spring Data JPA进行数据库操作
- 理解MVC架构模式在Spring Boot中的应用
- 掌握Thymeleaf模板引擎的使用
- 学会构建RESTful API
- 掌握数据验证和异常处理
- 了解全栈开发的最佳实践

## 🛠️ 技术栈

- **后端框架**: Spring Boot 3.5.7
- **Java版本**: Java 21
- **数据库**: MySQL 8.0
- **ORM框架**: Spring Data JPA
- **模板引擎**: Thymeleaf
- **构建工具**: Maven
- **开发工具**: IntelliJ IDEA

   - 安装Java 21 JDK
   - 安装MySQL 8.0
   - 安装Maven
   - 安装IntelliJ IDEA

2. **项目初始化**
   ```bash
   # 克隆或创建项目
   git clone [项目地址]
   cd SpringWeb

   # 导入项目到IDE
   # 配置数据库连接
   # 运行项目
   ```

3. **学习路径**
   - 按照教程顺序逐步学习
   - 每个阶段都有对应的代码示例
   - 建议边学边实践

## 📖 学习建议

### 循序渐进
- 不要跳过任何阶段，每个阶段都是后续学习的基础
- 理论学习与实践相结合，多动手编写代码

### 深入理解
- 不仅要知其然，更要知其所以然
- 查阅Spring Boot官方文档深入了解原理

### 实践为主
- 每学完一个阶段都要实际操作
- 尝试修改代码，观察效果
- 遇到问题先思考，再查阅资料
