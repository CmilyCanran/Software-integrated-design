# Spring Boot Web 开发完整教程

## 📚 教程概述

本教程采用**需求驱动、先前端后后端**的教学方法，通过开发一个完整的服装销售系统，帮助初学者循序渐进地掌握现代Web开发技术。

## 🎯 教程特色

### ✨ 对新手友好
- **先前端后后端**：先看到界面效果，增强学习信心
- **需求驱动**：每个功能都从实际需求出发
- **循序渐进**：从简单功能到复杂功能
- **立即可见**：每个阶段都能看到实际效果

### 🏗️ 现代化技术栈
- **前端**：Vue 3 + Element Plus + Vue Router + Pinia
- **后端**：Spring Boot 3 + Spring Security + JPA
- **数据库**：MySQL
- **架构**：前后端分离

### 🛍️ 实战项目
开发一个完整的**服装销售系统**，包含：
- 用户登录注册
- 商品展示与管理
- 购物车功能
- 订单管理
- 管理员后台

## 📋 教程结构

### 🎯 教学方式：功能驱动，前后端编写不分离

每个功能模块都按照以下流程教学：
1. **需求分析** - 明确功能目标
2. **前端开发** - 创建用户界面
3. **后端开发** - 实现对应API
4. **功能联调** - 前后端集成测试

## 🎯 学习路径建议

### 🌟 零基础初学者
1. **先学前端**：从第二阶段开始，先掌握Vue.js基础
2. **再学后端**：第五阶段学习Spring Boot
3. **最后集成**：第六阶段掌握前后端协作

### 🔄 有基础的学习者
1. **快速浏览**：第一阶段快速过一遍项目结构
2. **重点实践**：选择感兴趣的模块深入学习
3. **完整项目**：按照需求完成整个系统

### 💡 学习技巧
- **边学边做**：每个步骤都要实际编写代码
- **先跑通再优化**：不要追求完美，先让功能跑起来
- **多看控制台**：学会查看浏览器和服务器日志
- **善用调试**：掌握断点调试技巧

## 🛠️ 环境要求

### 必需软件
- **Node.js 16+** (前端开发)
- **Java 17+** (后端开发)
- **MySQL 8.0+** (数据库)
- **Maven 3.6+** (Java项目管理)

### 推荐工具
- **VS Code** (前端开发)
- **IntelliJ IDEA** (后端开发)
- **Postman** (API测试)
- **MySQL Workbench** (数据库管理)

### 1. 环境检查
```bash
# 检查 Node.js
node -v
npm -v

# 检查 Java
java -version

# 检查 Maven
mvn -version
```

### 2. 开始学习
1. 阅读项目环境准备
2. 按照教程顺序逐步学习
3. 每完成一个阶段都要实际运行测试

### 3. 遇到问题
- 查看对应章节的常见问题
