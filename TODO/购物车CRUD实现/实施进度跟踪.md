4. **异常处理** - 使用合适的异常类型

### 最佳实践
1. **逐步测试** - 每完成一个类就进行编译测试
2. **及时沟通** - 遇到问题立即询问
3. **代码审查** - 每个步骤完成后我会检查代码质量
4. **性能考虑** - 注意避免N+1查询等问题

3. **依赖注入** - 确保@Autowired或构造器注入正确
4. **事务管理** - 读写操作使用不同的事务配置
